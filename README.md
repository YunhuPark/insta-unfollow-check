# 🕵️‍♀️ 인스타 맞팔 판독기 (Instagram Unfollower Checker)

### "로그인 없이, 가장 안전하게 확인하세요."

인스타그램 계정 해킹이나 정지 걱정 없이, 내 정보 다운로드 파일(JSON)만으로 '나를 안 보는 사람'을 찾아주는 도구입니다. 서버로 그 어떤 데이터도 전송되지 않는 **100% 클라이언트 사이드(Client-side)** 앱입니다.

## 👉 [바로 사용해보기 (링크 클릭)](https://20202971-yh.github.io/insta-unfollow-check/)

---

## ✨ 주요 기능
- **🔒 보안 100%**: 아이디/비밀번호 입력 불필요.
- **📂 JSON 분석**: 인스타그램 공식 데이터 활용.
- **⚡ 즉시 분석**: 파일 업로드 즉시 결과 확인.
- **📱 모바일 최적화**: 스마트폰에서도 완벽 작동.

## 🛠 사용 방법
1. 인스타그램 앱에서 `내 활동` > `내 정보 다운로드` 요청 (JSON 형식 필수!)
2. 메일로 받은 파일 중 `followers.json`과 `following.json` 준비.
3. [사이트 접속](https://20202971-yh.github.io/insta-unfollow-check/) 후 파일 업로드.
4. 분석 버튼 클릭!
